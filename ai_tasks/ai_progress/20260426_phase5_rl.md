## 已创建的文件

### 核心模块
- `src/rl/__init__.py` - 模块初始化，导出 PolicyModel, ValueNetwork, RolloutStorage, LatentVecEnv, PPOLearner
- `src/rl/policy_model.py` - 策略网络，基于 IDM 架构，输出 Gaussian 动作分布
- `src/rl/value_network.py` - Value 函数网络，共享 Transformer 编码器
- `src/rl/storage.py` - RolloutStorage，循环缓冲区，支持 feed-forward mini-batch 生成
- `src/rl/vec_env.py` - LatentVecEnv，支持真实 latent cache 和 DummyVecEnv 测试
- `src/rl/ppo_learner.py` - PPOLearner，完整 PPO 训练逻辑
- `src/rl/train_rl.py` - Hydra 训练入口

### 配置文件
- `configs/rl_default.yaml` - 默认 RL 配置

## 数据源

- 现有 latent cache: `datasets/ai2thor/2026-04-22_14-50-39/manifest.latents.cfm_dinov2m.pt`
  - 40000 张图像 latent
  - 200 个 episode
  - 每个 episode ~200 步
  - Latent 维度: 16 patches x 384 token_dim = 6144

## 运行方式

```bash
# 默认配置（使用真实 latent cache）
uv run python src/rl/train_rl.py

# 自定义配置
uv run python src/rl/train_rl.py \
  rl.num_iterations=100 \
  rl.num_envs=16 \
  rl.num_steps=128 \
  rl.num_epochs=10 \
  rl.device=cuda

# 测试模式（Dummy 环境，无数据依赖）
uv run python src/rl/train_rl.py \
  rl.manifest_path=/nonexistent \
  rl.latent_cache_dir=/nonexistent
```
